# Инструкция по запуску проекта

Для проверки работы фронтенда и бэкенда (включая логин, регистрацию и т.д.) рекомендуется запускать проект через Docker. Это гарантирует, что все компоненты (база данных, сервер, интерфейс) будут настроены правильно и работать согласованно.

## Требования

Перед запуском убедитесь, что на компьютере установлены:
1.  **Docker Desktop** (для Windows/Mac) или **Docker Compose** (для Linux).
    *   Скачать: [https://www.docker.com/products/docker-desktop/](https://www.docker.com/products/docker-desktop/)

## Запуск проекта

1.  **Распакуйте архив** с проектом (или скачайте исходный код).
2.  Откройте терминал (PowerShell или командную строку) и перейдите в папку с проектом.
3.  Запустите следующую команду для сборки и запуска:

    ```bash
    docker-compose up --build
    ```

    *Первый запуск может занять несколько минут, так как скачиваются необходимые образы и устанавливаются зависимости.*

4.  Дождитесь сообщения в терминале, что веб-сервер запущен.

## Доступ к системе

После успешного запуска проект будет доступен по следующим адресам:

*   **Фронтенд (Сайт):** [http://localhost:3000](http://localhost:3000)
*   **Бэкенд (Админка):** [http://localhost:8000/admin](http://localhost:8000/admin)
*   **API:** [http://localhost:8000/api](http://localhost:8000/api)

## Создание тестовых данных (Опционально)

Чтобы не заполнять базу данных с нуля, вы можете создать тестовых пользователей (Агента и Партнера) и заявку:

1.  Убедитесь, что проект запущен.
2.  Выполните команду в терминале:

    ```bash
    docker-compose exec backend python manage.py shell -c "exec(open('scripts/seed_test_data.py').read())"
    ```

3.  После выполнения скрипта вы сможете войти с следующими данными:

    *   **Агент (Клиент):** `agent@lidergarantpanel.com` / `Admin123!`
    *   **Партнер (Банк):** `partner@lidergarantpanel.com` / `Admin123!`
    *   **Админ (Админ):** `admin@lidergarantpanel.com` / `Admin123!`

## Создание Администратора

Если вам нужен полный доступ к админ-панели:

1.  Откройте **новое окно терминала**.
2.  Выполните команду:

    ```bash
    docker-compose exec backend python manage.py createsuperuser
    ```

3.  Следуйте инструкциям (введите имя и пароль).

Теперь вы можете войти на [http://localhost:3000](http://localhost:3000) или [http://localhost:8000/admin](http://localhost:8000/admin) с этими данными.

## Остановка проекта

Чтобы остановить проект, нажмите `Ctrl+C` в окне терминала, где запущен Docker, или выполните команду:

```bash
docker-compose down
```

## Возможные проблемы

*   **Порты заняты:** Если вы видите ошибку `Bind for 0.0.0.0:3000 failed: port is already allocated`, значит этот порт занят другой программой. Освободите порт или измените его в файле `docker-compose.yml`.
*   **Docker не запущен:** Убедитесь, что приложение Docker Desktop запущено и работает.
