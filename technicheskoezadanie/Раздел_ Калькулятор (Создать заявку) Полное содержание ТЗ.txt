Раздел: Калькулятор (Создать заявку)
Назначение: Единый интерфейс для расчета стоимости продуктов, выбора банка и создания заявки.
1. Главное меню (Вкладки продуктов)
В верхней части калькулятора расположен переключатель продуктов:
1. Гарантии (БГ)
2. Кредиты
3. ВЭД
4. Лизинг
5. Страхование
6. Спецсчета
7. РКО
8. Тендерное сопровождение
2. Формы продуктов (Содержание вкладок)
Вкладка 1: Гарантии (Банковская гарантия)
Поля ввода:
* ИНН организации: (В окошке ввести - Название, ИНН или ОГРН).
   * Логика: При введении ИНН автоматически подтягивается информация из ЕГРЮЛ (Наименование, ИНН, ОГРН, юрадрес).
* Тип банковской гарантии (Выпадающий список):
   * Обеспечение заявки
   * Обеспечение исполнения контракта
   * Возврат аванса
   * Гарантийные обязательства
   * Гарантии оплаты товара
   * Таможенные гарантии
   * На возмещение НДС
* Закон исполнения торгов (Выпадающий список):
   * 44-ФЗ
   * 223-ФЗ
   * 185-ФЗ (615-ПП)
   * КБГ (Коммерческие)
* № закупки (В окошке - Номер закупки):
   * Логика: При введении номера закупки автоматически подтягивается информация из ЕИС через API (начальная и предложенная цена контракта, сумма БГ, дата гарантии, наименование заказчика).
Чек-боксы (Галочки/Переключатели):
* Наличие аванса
* Является переобеспечением
* Закрытый конкурс/аукцион
* Единственный поставщик
* Без размещения в ЕИС
Блок: Данные заказчика
* ИНН: (Окошко - Название, ИНН или ОГРН компании)
Блок: Финансовые параметры
* Начальная цена контракта, руб: (Поле ввода 0,00)
* Предложенная цена контракта, руб: (Поле ввода 0,00)
   * Опция: Под окошком галочка — "Торги еще не проведены". (При нажатии поле "Предложенная цена" выключается/меняет цвет).
* Сумма банковской гарантии, руб: (Поле ввода цифры с копейками 0,00)
* Срок банковской гарантии:
   * с (Календарь) по (Календарь)
Дополнительная опция:
* Кредит на пополнение оборотных средств:
   * Галочка: "Клиенту нужен кредит".
   * Текст: "Автоматически создадим и рассмотрим заявку на кредит данному Клиенту у вас в кабинете в разделе 'Кредиты на пополнение оборотных средств'".
Вкладка 2: Кредиты
В этом разделе предусмотрено переключение или выбор подтипа кредита.
Подтип А: Кредит на исполнение контракта
* ИНН организации-заявителя: (Автозаполнение из ЕГРЮЛ)
* № закупки: (Автозаполнение из ЕИС)
* Закон исполнения торгов: 44-ФЗ, 223-ФЗ, 185-ФЗ, КБГ.
* Чек-боксы: Наличие аванса, Закрытый конкурс.
* Данные заказчика: ИНН.
* Цены: Начальная цена, Предложенная цена (галочка "Торги еще не проведены").
* Параметры кредита:
   * Сумма кредита, руб.
   * Срок финансирования (с... по...).
Подтип Б: Корпоративный кредит
* ИНН организации-заявителя: (Автозаполнение из ЕГРЮЛ)
* Тип кредита (Выпадающий список):
   * Разовый кредит
   * Невозобновляемая Кредитная Линия (НКЛ)
   * Возобновляемая кредитная линия (ВКЛ)
   * Кредит на исполнение контракта
* Сумма кредита: (Ввод значения)
* Дата окончания: (Календарь)
Вкладка 3: ВЭД (Валютные операции)
* ИНН организации-заявителя: (Автозаполнение из ЕГРЮЛ)
* Сумма платежа: (Ввод значения)
* Валюта: (Выпадающий список валют + кнопка "Выбрать")
* Страна платежа: (Выпадающий список стран. Первая — Россия, далее по алфавиту)
* Кнопка действия: "Подать заявку"
Вкладка 4: Лизинг
* ИНН организации-заявителя: (Автозаполнение из ЕГРЮЛ)
* Тип предмета лизинга/кредита (Примечание: в ТЗ названо "Тип кредита", но контекст Лизинга):
   * Разовый кредит / Линии (Вероятно, здесь ошибка в исходнике и должны быть типы лизинга: Авто, Оборудование и т.д., но оставляю как в файле СЗ - Лизинг.csv: Разовый кредит, НКЛ, ВКЛ, Кредит на исполнение).
* Сумма: (Ввод значения)
* Дата окончания: (Календарь)
Вкладка 5: Страхование
* ИНН организации-заявителя: (Автозаполнение из ЕГРЮЛ)
* Вид страхования (Главный переключатель):
   1. Персонал
   2. Транспорт
   3. Имущество
   4. Ответственность
Детализация продуктов (появляется в зависимости от Вида):
* Если Персонал: Добровольное медицинское страхование (ДМС), Страхование критических заболеваний, Страхование несчастных случаев, Комплексное страхование в поездках.
* Если Транспорт: ОСАГО юридических лиц, Комплексное страхование автопарков, Страхование специальной техники, Страхование ответственности перевозчика.
* Если Имущество: Страхование объектов строительства, Страхование грузов и перевозок, Страхование имущества компаний, Страхование перерывов деятельности.
* Если Ответственность: Страхование гражданской ответственности, Страхование опасных объектов, Страхование профессиональных рисков, Страхование ответственности за качество.
Параметры:
* Сумма страхования, руб: (Ввод значения)
* Срок договора, мес: (Выбор: 1-16 месяцев)
* Кнопка: "Расчет страхования"
Вкладка 6: Факторинг
* ИНН организации-заявителя: (Автозаполнение из ЕГРЮЛ)
* ИНН контрагента: (Ввод ИНН)
* Тип факторинга (Выпадающий список):
   * Классический факторинг
   * Закрытый Факторинг
   * Закупочный факторинг
* Сумма Финансирования: (Ввод значения)
* Срок финансирования: (Календарь)
Вкладка 7: РКО и спецсчета
* ИНН организации-заявителя: (Автозаполнение из ЕГРЮЛ)
* Тип (Выпадающий список):
   * РКО
   * Спецсчет
* Кнопка: "Выбрать банк"
Вкладка 8: Тендерное сопровождение
* ИНН организации-заявителя: (Автозаполнение из ЕГРЮЛ)
* Вариант сопровождения (Выпадающий список):
   * Разовое сопровождение
   * Тендерное сопровождение под ключ
* Тип закупки (Ввод значения):
   * Госзакупки по 44-ФЗ
   * Закупки по 223-ФЗ
   * Имущественные торги
   * Коммерческие закупки
* Закупки в отрасли: (Ввести интересующую отрасль закупок)
* Кнопка: "Получить предложение"
3. Результаты расчета (Нижняя часть экрана)
После нажатия кнопки "Получить предложение" / "Рассчитать" / "Выбрать банк":
Логика отображения:
* Отбор идет по критериям, суммам и стоп-факторам, прописанным в условиях банков.
* Сначала показываются те, кто готов (зеленые/доступные).
* Ниже (в свернутом виде) те, кто отказал (если развернуть, можно посмотреть причину отказа).
Таблица предложений (Список банков):
* Столбец 1: Наименование Банка / МФО / Страховой компании.
   * Спец. условие: Внизу списка всегда отображается строка "Лидер-Гарант, Индивидуальные условия".
* Столбец 2: Тариф (%, руб.) / Платеж.
* Столбец 3: Скорость принятия решения (Высокая, Средняя, Низкая).
* Столбец 4: Выбрать (Чек-бокс / Галочка).
   * Логика: Если выбрать "Лидер-Гарант", остальные галочки убираются.
Футер (Подвал):
* Кнопка: Создать заявку (основное действие).
* Кнопка: Очистить форму.
* Дисклеймер: "Приведенные расчеты стоимости являются предварительными и не являются публичной офертой."
* Навигация: Стрелочка "Наверх" (справа внизу).
